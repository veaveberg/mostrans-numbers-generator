# MosTrans Numbers Generator

React + TypeScript + Vite app for generating tram number sticker PDFs from CSV data.

## Dev

- `npm install`
- `npm run dev`

## Build

- `npm run build`
- `npm run preview`

## GitHub Pages

This repo includes a workflow that builds the app on pushes to `main` and deploys `dist/` to GitHub Pages.

After the first deploy, the app should be available at:

- `https://veaveberg.github.io/mostrans-numbers-generator/`

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
